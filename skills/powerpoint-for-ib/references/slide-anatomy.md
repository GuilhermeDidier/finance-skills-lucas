# Anatomia de um Slide de Banco

> A estrutura de um slide bem-feito de IB. Cada slide é um argumento, não um depósito de dados.

## As partes de um slide
```
┌───────────────────────────────────────────────┐
│ ACTION TITLE (o "takeaway" — uma frase-conclusão)│  ← a mensagem
├───────────────────────────────────────────────┤
│  [subtítulo opcional / contexto]                │
│                                                 │
│  CORPO: gráfico / tabela / bullets que          │  ← a evidência
│  PROVAM o action title                          │
│                                                 │
├───────────────────────────────────────────────┤
│ Source: ... (fonte) | nota de rodapé            │  ← credibilidade
└───────────────────────────────────────────────┘
```

## Action title (a regra de ouro)
- O título diz a **conclusão**, não o tópico. ❌ "Múltiplos de mercado" → ✅ "Os múltiplos do setor se expandiram 2x desde 2022, sustentando uma janela atrativa de venda".
- Teste: **lendo só os action titles em sequência**, o cliente entende a história inteira (o "horizontal flow" do deck).
- Uma frase, específica, com o **"so what"**. Evite títulos genéricos/descritivos.

## Uma mensagem por slide
- Cada slide prova **um** ponto. Se há dois argumentos, são dois slides.
- O corpo (gráfico/tabela) deve **provar o título** — se não prova, ou o título ou o corpo está errado.
- "**So what?**": para todo número/gráfico, responda por que importa. Dado sem interpretação não é argumento.

## Corpo: dado vira argumento
- Prefira **um gráfico/tabela bem escolhido** a paredes de bullets.
- **Destaque** o que importa (cor, círculo, seta, negrito) — guie o olho para o número que sustenta o título.
- Bullets: paralelos, concisos, no máximo ~2 níveis; não frases longas. Regra prática "menos é mais".
- **Call-outs** (caixas/setas) conectam o dado à conclusão.

## Rodapé e fontes
- **Sempre** cite a fonte (FactSet, Capital IQ, filings, research, company materials) — credibilidade e auditabilidade.
- Notas de rodapé explicam premissas/ajustes (ex.: "EBITDA ajustado por não-recorrentes"; ver skill valuation/financial).
- Numere as páginas; marque **"Confidential" / "Draft"** quando aplicável.

## Fluxo de leitura
- O olho ocidental lê **Z** (cima-esquerda → direita → baixo). Coloque o mais importante no topo-esquerda.
- Consistência de posição entre slides (títulos, rodapés, paginação no mesmo lugar).
