# Exemplo trabalhado — Dois slides de pitchbook (descritos)

> Como os princípios viram slides reais. Descrição em ASCII (já que é PowerPoint) de um slide de **football field** e um de **positioning**, com action titles.

## Slide 1 — Football Field (valuation)
```
┌──────────────────────────────────────────────────────────────────────┐
│ A triangulação aponta valor de R$52–57/ação — ~10% acima do preço      │ ← action title (conclusão)
│ atual de R$50, com upside sustentado por crescimento e desalavancagem  │
│ ($ por ação)                                                           │
│                                                                        │
│  DCF (perpetuidade→exit)        ███████████  52,1 ─ 57,4               │
│  Trading comps (EV/EBITDA)     ████████████  47,2 ─ 58,4               │
│  Transaction comps             N/A (não é venda)                        │
│  LBO (floor)                 ██████  43 ─ 49                            │
│  52-week range            █████████  41 ─ 55                            │
│                          │           ┊ preço atual R$50,00              │
│                       40   45   50   55   60                            │
│                                                                        │
│  Faixa selecionada: R$52–57  |  EV/EBITDA implícito 6,5–7,0x            │
├──────────────────────────────────────────────────────────────────────┤
│ Source: FactSet, filings da companhia, estimativas BAM. EBITDA ajustado.│
└──────────────────────────────────────────────────────────────────────┘
```
**Por que funciona:** o título dá a conclusão (não "Football Field"); as barras provam a faixa; o preço atual ancora; múltiplos implícitos checam razoabilidade; fonte citada. → dados da skill valuation-modeling.

## Slide 2 — Positioning / Equity Story
```
┌──────────────────────────────────────────────────────────────────────┐
│ A TechBR combina liderança em software B2B com receita 95% recorrente — │ ← action title
│ um perfil que atrai tanto strategics quanto sponsors                    │
│                                                                        │
│  ┌─ Liderança ──────┐  ┌─ Recorrência ────┐  ┌─ Crescimento ────┐      │
│  │ #1 em nicho X     │  │ 95% ARR           │  │ +22% receita a.a. │      │
│  │ market share 34%  │  │ NRR 118%          │  │ Rule of 40 = 47   │      │
│  └───────────────────┘  └───────────────────┘  └───────────────────┘     │
│                                                                        │
│  ● Switching costs altos (lock-in via integração) → moat defensável    │
│  ● TAM de R$X bi, penetração <15% → pista longa de crescimento         │
│  ● Margem EBITDA expandindo (operating leverage)                       │
├──────────────────────────────────────────────────────────────────────┤
│ Source: company materials, FactSet. NRR = net revenue retention.        │
└──────────────────────────────────────────────────────────────────────┘
```
**Por que funciona:** uma mensagem (o equity story), três pilares visuais que a sustentam, métricas com "so what", e o moat conectado à tese (→ skill corporate-finance / valuation `moat.md`).

## O teste do "horizontal flow"
Lendo só os dois action titles em sequência:
> "Vale R$52–57 (10% de upside)... porque combina liderança + recorrência que atraem compradores."
A história já se entende **só pelos títulos** — esse é o sinal de um deck bem construído.
